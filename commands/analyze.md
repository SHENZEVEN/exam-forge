# /exam-forge analyze — 真题分析

## 做什么
分析真题的题型结构、知识点分布、难度曲线、出题风格，生成分析报告。

## 分析维度
1. **学科类型判定** — 理工计算型 / 文商概念型 / 语言考试型
2. **题型结构** — 每种题型的数量、分值、占比
3. **逐题提取** — 题号、题型、知识点、难度
4. **知识点分布** — 高频考点、知识点-题型映射
5. **难度曲线** — 简单/中等/困难比例、送分题/压轴题位置
6. **出题风格** — 语言、措辞、陷阱倾向、连环题模式
7. **重点覆盖** — 老师重点的覆盖情况（如有）

## 输出
分析报告展示给用户确认。
