# /exam-forge generate — 模拟卷生成

## 做什么
基于分析结果，生成风格复刻的全新模拟卷。

## 产出
- 完整试卷（题目 + 分值 + 考试时间）
- 逐题答案 + 解题步骤 + 关键 insight + 常见错误 + 关联知识点
- 知识点覆盖报告（原卷 vs 模拟卷）
- 分层复习建议（60分 / 80分 / 90分+）

## 硬约束
1. 绝不复用原题（新数据、新场景、新角度）
2. 风格复刻（措辞、编号、题型顺序一致）
3. 难度偏差 ≤ 5%
4. 知识点覆盖 ≥ 原卷覆盖 + 老师重点
5. 不编造新题型

## 输出路径
`outputs/模拟卷（一）-[课程名].md`

## 下一步
→ `/exam-forge export md` 或 `/exam-forge export docx`
→ 不满意可 `/exam-forge regenerate`
